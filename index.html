🚀 Milestone Projects

Four projects that compound on each other. Each one becomes a portfolio piece on your GitHub. Recruiters don't care about certificates as much as they care about repos with real, working code.

Phase 1 · Weeks 1–6

Python Recon Toolkit + Kali Homelab

Set up a virtualized lab and build a modular Python reconnaissance tool that automates everything you've learned.

Deliverables:
  • VirtualBox lab: Kali + Metasploitable 2 with snapshots
  • Python CLI tool with argparse: host discovery, port scan, service enum, banner grab
  • JSON output schema for integration with future tools
  • GitHub repo with README, usage examples, screenshots
Phase 2 · Weeks 7–12

Web Vulnerability Scanner + VA Report

Build a basic web vuln scanner that detects SQLi, XSS, and open redirects. Use it against OWASP Juice Shop. Produce a professional VA report.

Deliverables:
  • Python scanner: crawls URLs, sends payloads, parses responses
  • Detection rules for 3 vulnerability classes
  • Run against OWASP Juice Shop and document findings
  • 10-page VA report: executive summary, methodology, findings, CVSS scores, remediation
Phase 3 · Weeks 13–18

Full Black-Box Penetration Test Report

Complete a HackTheBox Medium-difficulty machine from zero to root, with no walkthrough. Document the full kill chain as a professional pentest report.

Deliverables:
  • 1 HackTheBox Medium box, no hints, time-boxed to 1 day
  • Full nmap + service enumeration notes
  • Initial access exploit documented step-by-step
  • Privilege escalation vector with screenshots
  • 15+ page pentest report (PDF) following PTES structure
Phase 4 · Weeks 19–24

Active Directory Red Team Engagement

The capstone. Build a 3-machine AD lab, execute a full red team engagement from initial access to Domain Admin, and deliver a report with MITRE ATT&CK mapping.

Deliverables:
  • Home lab: 1 Windows Server 2022 DC + 2 Windows 10 clients + Kali attack box
  • Engagement: Kerberoasting → lateral movement → DCSync → Domain Admin
  • Sliver C2 implant deployment and beaconing
  • Full engagement report with MITRE ATT&CK technique mapping
  • 15-minute video walkthrough uploaded to YouTube (unlisted is fine)

⏰ Daily Routine

This is a 4-hour-per-day commitment for 168 days. That's 672 hours of focused work. The schedule below is non-negotiable. Block these hours on your calendar and defend them like your job depends on it — because it does.

📅 Monday–Friday (4 hours daily)

60 minTheory: watch the day's video / read the day's docs. Take handwritten notes.
90 minGuided coding: follow along with the lab/tutorial. Don't just watch — type the commands.
60 minProject work: apply what you learned to your milestone project. No tutorials.
30 minAlgorithms drill: 1 NeetCode problem. Review solution even if you solved it.

🗓️ Saturday (Deep Work)

4–6 hrsOne block of focused work on the weekly project milestone. No interruptions. No social media.
30 minEnd-of-day review: what did you build? What broke? What did you learn? Write it down.
—Push commit to GitHub even if it's WIP. Maintain the green-square streak.

🗓️ Sunday (Review & Rest)

60 minReview the week's notes. Update your portfolio. Plan the next week on paper.
60 minRe-do 1 algorithm problem from earlier in the week to verify retention.
RestMental recovery is part of the process. Burnout is the enemy. Step away from the screen.
