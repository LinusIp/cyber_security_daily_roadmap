🚀 Projects

These projects are your portfolio. Each one demonstrates a different skill to employers. Complete at least 4 of 6 before day 168. All code goes on GitHub with proper READMEs.

Python · Networking

Project 1: Custom Network Scanner

Build a Python CLI tool that performs SYN scans, service detection, and OS fingerprinting. Use Scapy for packet crafting and threading for performance. Output results as JSON and a styled HTML report.

Deliverables:
  • Working scanner with argparse CLI
  • SYN scan, connect scan, UDP scan modes
  • Service version detection
  • JSON + HTML report output
  • GitHub repo with README, requirements.txt, tests
Web Security · Automation

Project 2: Automated XSS Fuzzer

Build a tool that takes a URL, finds all input parameters (via crawling or sitemap), and tests each for reflected XSS using a curated payload list. Integrate with headless browser for DOM-based XSS detection.

Deliverables:
  • Parameter discovery (forms, URL params, headers)
  • Reflected XSS detection with context-aware payloads
  • DOM-based XSS detection via Playwright/Selenium
  • CSV report with payload, parameter, evidence
  • GitHub repo with documentation
AD · Red Team

Project 3: AD Attack Automation Script

Build a Python script that automates common AD enumeration and attack steps: BloodHound data collection, Kerberoasting, AS-REP Roasting, DCSync detection check, and GPO abuse enumeration. Output a structured report.

Deliverables:
  • Automated domain enumeration
  • Kerberoast + AS-REP roast with hashcat integration
  • Privilege escalation path identification
  • MITRE ATT&CK mapping for each technique
  • GitHub repo — mark as private if sensitive
Blue Team · SIEM

Project 4: Home Lab SOC Dashboard

Set up ELK stack or Splunk free in your home lab. Ingest logs from your Kali VM, Windows VM, and router. Create dashboards for: failed login attempts, network anomalies, process execution, and file modifications.

Deliverables:
  • Working ELK/Splunk stack with 3+ log sources
  • 5 detection rules with alerts
  • 3 dashboards (overview, auth, network)
  • Writeup: architecture diagram, installation steps, screenshots
Cloud · DevSecOps

Project 5: AWS Security Assessment Tool

Build a Python tool using boto3 that audits an AWS account for: public S3 buckets, overly permissive IAM policies, unencrypted EBS volumes, open security groups, and CloudTrail configuration issues.

Deliverables:
  • 10+ AWS security checks
  • CSV + HTML report with severity ratings
  • Remediation suggestions for each finding
  • GitHub repo with CI/CD pipeline example
Capstone · Full Engagement

Project 6: End-to-End Pentest Report

Select 3 VulnHub machines or build your own lab. Conduct a full pentest engagement from scoping to final report. This is the project you show in interviews. Make it look like a real consulting deliverable.

Deliverables:
  • Rules of Engagement document
  • Full pentest report (15+ pages)
  • Executive summary + technical findings
  • CVSS scores, remediation priorities, proof of concepts
  • PDF on GitHub — this is your crown jewel
